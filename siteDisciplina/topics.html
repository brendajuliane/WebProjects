Introdução

Este tópico aborta de maneira geral o funcionamento da web, com foco nas interações navegador-servidor, e contextualiza a Programação para Web. Em linhas gerais, descreve os passos executados a cada requisição feita por um navegador a um servidor, interpretar uma mensagem HTTP

HTML

Apresenta o básico necessário sobre Hypertext Markup Language (HTML) e estrutura de documentos HTML. Neste tópico vemos exemplos de elementos HTML (parágrafos, títulos, imagens, hyperlinks, tabelas, listas e iFrames) e formas de estruturar o conteúdo em layouts bem definidos.

CSS

Na disciplina vemos também definições, sintaxe e formas de utilização do Cascade Style Sheets (CSS). A partir disso podemos construir folhas de estilo que definam a aparência de um documento HTML e compreender a formatação que será aplicada a um elemento HTML por regras CSS já criadas.

JavaScript

Este tópico aborda a programação no front-end com JavaScript. Vemos definições, sintaxe, instruções e o objeto DOM. Além disso, a detecção/reação a eventos e a manipulação de formulários HTML via JS também são abordados.

PHP

Neste tópico, Orientação a Objetos, acesso a Bancos de Dados, Cookies, e Sessões são abordados. Além disso, a comunicação assíncrona de Dados entre front-end e back-end é bordada.
